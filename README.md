# Backend Lab 1 - Tasks API

A simple REST API built with **Node.js**, **Express**, and **MySQL** using **MVC architecture**.

## Features

* Get all tasks
* Get task by ID
* Create task
* Update task
* Delete task

## Technologies

* Node.js
* Express.js
* MySQL
* MVC Architecture

## API Endpoints

GET /tasks
GET /tasks/:id
POST /tasks
PUT /tasks/:id
DELETE /tasks/:id

## Installation

Clone the repository

git clone https://github.com/Yousef-martaa/backend-lab1.git

Install dependencies

npm install

Run the server

node app.js

Server runs on:

http://localhost:3000
